YOUR CONTROL GUIDE

📧 ALLY SIGN-UPS

When a new ally signs the pledge, the form sends an email to troungnvo@icloud.com. Check that email to see new ally names, their city, age, and school. Then add them to the Ally Wall using the editor below.

📦 CARE PACKAGE REQUESTS

Care package applications also go to troungnvo@icloud.com. Each email has the kid's name, age, disability, city, and parent email. Reply to arrange the care package delivery.

⭐ SPOTLIGHT — HOW TO CHANGE IT

Open spotlight.html and find the section that says THIS MONTH'S SPOTLIGHT. Change the name, quote, story, condition, and tags. Save and push to GitHub to go live.

🚀 HOW TO PUSH UPDATES

After making changes, open Terminal, go to the over folder, then run: git add -A → git commit -m "update" → git push. Changes go live in 1–2 minutes.
